# Backend de Adopción de Gatitos
Bienvenido al repositorio del backend para el sistema de Adopción de Gatitos. Este proyecto es parte de una arquitectura de microservicios que proporciona la funcionalidad necesaria para gestionar y servir la información de los gatos disponibles para adopción.

Descripción
Este backend está diseñado para proporcionar una API RESTful que interactúa con una base de datos para manejar la información sobre los gatos disponibles para adopción. 

Características
Endpoints de Gatitos: Ofrece endpoints para obtener la lista de gatos y obtener detalles de un gato específico.
Base de Datos: Utiliza MongoDB para almacenar información sobre los gatos, incluyendo nombre, imagen, descripción, y otros detalles relevantes.

Tecnologías Utilizadas
Node.js: Entorno de ejecución para JavaScript en el backend.
Express: Framework para construir la API RESTful.
Mongoose: Biblioteca para interactuar con MongoDB.
MongoDB: Base de datos NoSQL para el almacenamiento de la información de los gatos.
